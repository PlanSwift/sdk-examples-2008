History
 Original Release Date 7/25/2008

General requirements:
  Office 2007 (older versions may still work). Import Excel, Word, and Outlook type libraries using <Component><Import Component> on the main menu.
  Word Example will look for Work Order.dotx in the templates folder.
  AutoCad 2006 (again, older versions may work). Import type library as above.
